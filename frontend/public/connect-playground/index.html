Connect Playground

Two independent apps, one user — test portable identity, OAuth connect, and grant flows.

Test checklist
  1. Log into the Runmesh dashboard on this origin, or paste a developer JWT below.
  2. Create demo apps (or pick existing ones) and confirm redirect URIs match the callback pages.
  3. Authenticate App A with your email and OTP code from wrangler logs ([connect-otp]).
  4. Authenticate App B with the same email — confirm the banner shows the same connect_user_id.
  5. Run Connect Google on App A, finish OAuth, land on callback-a.
  6. Run Grant scopes on App B — should reuse the Google connection (consent screen).
  7. Use List grants on each app and verify grants in Settings → Connect.
